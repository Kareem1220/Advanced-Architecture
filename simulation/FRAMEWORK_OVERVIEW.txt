CUSTOMIZATION OPTIONS
----------------------
Dataset Size:
  → Edit NUM_DOCS in run_all_experiments.py (line ~380)
  → Options: 10K (fast), 100K (standard), 1M+ (scalability)

Hardware Config:
  → Use HardwareConfig.get_config('baseline')
  → Or 'high_performance', 'low_power'
  → Or create custom configuration

Embedding Model:
  → Change model in EmbeddingGenerator
  → Default: 'sentence-transformers/all-MiniLM-L6-v2' (384D)
  → Alternative: 'all-mpnet-base-v2' (768D, more accurate)

TROUBLESHOOTING
---------------
Issue: "No module named 'numpy'"
  Fix: pip install -r requirements.txt

Issue: "GPU not available"
  Fix: Normal if no CUDA GPU; simulation still works

Issue: Out of memory
  Fix: Reduce NUM_DOCS to 10000

Issue: Tests fail
  Fix: cd simulation && python test_simulation.py
